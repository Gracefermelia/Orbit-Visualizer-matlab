# Orbit-Visualizer-matlab
ORBIT_VISUALIZER Interactive 3D Elliptical Orbit Visualizer (Earth centered, ECI frame) Drag sliders to change orbital elements in real time:    a (km), e, i (deg), RAAN Ω (deg), ω argument of periapsis (deg), ν true anomaly (deg).  Shows full orbit path, current spacecraft position, and Earth as a sphere
